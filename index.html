TypeScript

The birth of Typescript dates back to the year 2012, developed by Microsoft with Anders Hejilsberg as the main architect.

TypeScript is a superset of JavaScript.
What are the advantages of typescript?
  • Less Error Prone
  • More Human Readable
  • More Maintainable and Scalable
Important typescript concepts

Type Annotations

Example: 

				let name: string = "John";
    let age: number = 29;

Interfaces

Example: 

				interface Product {
					id: number;
					name: string;
					price: number;
				  }
				  
				  const product: Product = { id: 1, name: "T-shirt", price: 12 };

Generics

Example: 

              function identity<T>(value: T): T {
                return value;
              }
              
              const number = identity(5); 
              const string = identity("hello");

Classes

Example: 

              class User {
                name: string;
                age: number;
                constructor(name: string, age: number) {
                  this.name = name;
                  this.age = age;
                }
                greet() {
                  console.log(`Hello, my name is ${this.name}!`);
                }
              }
              const user = new User("Alice", 35);
              user.greet();

Union Types

Example: 

              let code: string | number;
              code = 123;   // OK
              code = "ABC"; // OK
              code = false; // Compiler Error

Type Inference

Example: 

              // This will automatically be inferred as 
              // a number type because it is assigned a numeric value.
              let age = 28;   // number

Type Aliases

Example: 

              // Creating aliases for simple data types
              type StringOrNumber = string | number;

              // Creating an alias for an object:
              type User = {
                id: StringOrNumber
                firstName: string;
                pointsCount: number;
              }
              const user = {
                id: 208
                firstName: 'Mike',
                pointsCount: 1000,
              };

Enums

Example: 

              enum ResourceType {
                BOOK,
                AUTHOR,
                FILM,
              }
              console.log(ResourceType.BOOK); // 0
              console.log(ResourceType.AUTHOR); // 1
              enum Direction {
                Up = 'Up',
                Right = 'Right',
                Down = 'Down',
                Left = 'Left',
              }
              console.log(Direction.Right); // Right
              console.log(Direction.Down); // Down

Type Assertions

Example: 

              const myCanvas = document.getElementById("main_canvas") as HTMLCanvasElement;
